## Зачем нужен парсер выписок
Для импорта транзакций требуется заполнить поля файла в определенном порядке, но выписки из разных банков отличаются. Привести их к единому формату поможет Парсер транзакций.

Парсер собирает данные из выписок, приводит под требования к импорту, позволяет указать категории, а также при наличии категорий контрагентов - находит контрагента по ИНН и автоматом проставляет нужную категорию.

### 1. Подготовка
Для получения файла парсера обратитесь в чат технической поддержки.

Скопируйте к себе на диск Google-таблицу receipt_parser. Это надо будет сделать 1 раз, дальше уже можно каждый раз просто перезаписывать вашу копию.

[[info type=custom hasIcon icon=info color=#E06823]]
Листы **Категории контрагентов** и **Категории** удалять нежелательно, это справочники которые облегчают работу. Их можно дополнять новыми категориями. Для автоматического проставления категорий контрагентов необходимо заполнить справочник (если таких нет, просто игнорируйте этот лист)
[[/info]]

![receipt_parser-1](/ru/images/receipt_parser/receipt_parser-1.png)

### 2. Загрузка выписок

Импортируйте файлы выписок как отдельные листы с именами:
- receipt_alfa
- receipt_vtb
- receipt_sber

### 3. Запуск парсинга

**1.** В верхнем меню нажмите кнопку `Парсинг выписок` -> Запустите парсинг нужной выписки / всех выписок (если запуск с этого аккаунта производится впервые - примите все нужные соглашения). Если кнопка не отображается - несколько раз перезагрузите страницу или закройте файл и откройте его заново.

[[gallery gap=12 layout=carousel]]
![receipt_parser-2](/ru/images/receipt_parser/receipt_parser-5.png)
![receipt_parser-3](/ru/images/receipt_parser/receipt_parser-6.png)
[[/gallery]]

**2.** Дождитесь генерации файлов с собранными транзакциями из нужных выписок.

**3.** Затем перейдите на лист `transactions_[нужный банк]` и скачайте csv файл. Его далее можно импортировать в модуле Транзакций.

![receipt_parser-3](/ru/images/receipt_parser/receipt_parser-7.png)


[[info type=custom hasIcon icon=info color=#E06823]]
Поля `categoryId, hour и minute` не заполняются автоматически, их нужно будет заполнить самостоятельно
[[/info]]

- **Категорию** можно выбрать в селекте из справочника либо ввести ID вручную. Если категория = контрагент, она проставится автоматически по ИНН контрагента.
- Если **время транзакции** не важно - просто "протащите" 12 и 00.
